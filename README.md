# Control Privacy Site

Repositorio exclusivo para hospedar a Politica de Privacidade do aplicativo Control.

## Publicar Pelo GitHub Pages

1. Crie um repositorio publico no GitHub.
   Sugestao: `control-privacy-policy`

2. Envie todos os arquivos desta pasta para a raiz do repositorio.

3. No GitHub, abra `Settings > Pages`.

4. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Salve.

6. Use esta URL no Google Play Console:
   `https://SEU_USUARIO.github.io/control-privacy-policy/`

## Arquivos

- `index.html`: Politica de Privacidade publica.
- `privacy-policy.html`: URL alternativa que redireciona para a pagina principal.
- `play-store-declarations.md`: guia interno para preencher o Play Console.
