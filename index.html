Politica de Privacidade - Control

Ultima atualizacao: 6 de julho de 2026

Esta Politica de Privacidade descreve como o aplicativo Control trata os dados do usuario. O Control e um aplicativo de controle pessoal de ponto, horas trabalhadas e banco de horas.

1. Dados Tratados

O Control armazena apenas os dados necessarios para o funcionamento do aplicativo:

  • Dados de cadastro local, como nome, e-mail e senha protegida por hash.
  • Dados de jornada, como horarios de entrada, saida, pausas, banco de horas e configuracoes de jornada.
  • Pergunta e resposta de recuperacao de acesso, protegidas por criptografia local.
  • Fotos ou imagens anexadas pelo proprio usuario aos registros de ponto.

O Control nao coleta localizacao, nao usa anuncios, nao usa analytics e nao envia dados automaticamente para servidores.

2. Armazenamento

Todos os dados ficam armazenados localmente no dispositivo do usuario.

Dados sensiveis usados para recuperacao sao criptografados localmente. A chave de criptografia e mantida no armazenamento seguro do sistema quando disponivel.

Backups exportados pelo usuario sao gerados em arquivo criptografado e compartilhados somente por acao manual do usuario.

3. Finalidade

Os dados sao usados exclusivamente para:

  • Registrar e calcular a jornada de trabalho.
  • Gerar relatorios, estatisticas e historico de ponto.
  • Permitir recuperacao de acesso local.
  • Permitir exportacao e restauracao de backup quando solicitado pelo usuario.

Os dados nao sao usados para publicidade, analytics, perfil comportamental ou venda a terceiros.

4. Compartilhamento

O Control nao vende, nao envia e nao compartilha dados automaticamente com terceiros.

O usuario pode compartilhar manualmente arquivos de backup, CSVs ou anexos usando os recursos de exportacao do proprio dispositivo. Nesse caso, o destino escolhido pelo usuario passa a ser responsavel pelo tratamento do arquivo compartilhado.

5. Permissoes Do Aplicativo

O Control pode solicitar permissoes para:

  • Camera: anexar fotos aos registros de ponto.
  • Fotos e imagens: selecionar imagens para anexos.
  • Notificacoes: enviar lembretes locais de registro de ponto.
  • Biometria: permitir desbloqueio local opcional.

As permissoes sao usadas apenas para as funcionalidades informadas.

6. Seguranca

O Control utiliza medidas de protecao compativeis com seu funcionamento local:

  • Armazenamento local no dispositivo.
  • Hash para senha de acesso.
  • Criptografia local para dados sensiveis e backups.
  • Autenticacao biometrica opcional, quando suportada pelo dispositivo.

Como os dados ficam no dispositivo, a seguranca tambem depende das protecoes do aparelho, como bloqueio de tela, biometria e backup seguro.

7. Retencao E Exclusao

Os dados permanecem no dispositivo ate que o usuario os exclua, remova a conta, desinstale o app ou limpe os dados do aplicativo pelo sistema operacional.

Arquivos exportados pelo usuario, como backups e CSVs, ficam fora do controle do Control apos o compartilhamento ou salvamento externo.

8. Direitos Do Usuario

O usuario pode acessar, corrigir, exportar e excluir seus dados locais usando as funcionalidades do aplicativo.

9. Criancas

O Control nao e destinado a criancas. O publico recomendado para o aplicativo e adulto, por se tratar de controle de jornada e trabalho.
